Abstract

Memory is critical for long-horizon robot manipulation characterized by stage ambiguity and partial observability: the robot may encounter visually similar observations at different stages of execution, yet the correct action depends critically on execution history. Existing memory-augmented approaches either blur discrete stage transitions through dense compression and token merging, or struggle to maintain relevant context over very long horizons. We propose event-driven keyframe memory, a plug-in memory framework that resolves stage ambiguity by precisely identifying and preserving event keyframes corresponding to task-relevant state transitions. Our keyframe detector combines kinematic signals with visual change detection, targeting timesteps where robot actions produce task-relevant state changes. Detected keyframes populate a keyframe memory bank whose compressed tokens are injected into current visual representation via cross-attention, providing the policy with explicit stage-progress evidence. We evaluate our method on real-world dual-arm long-horizon manipulation tasks with average trajectory durations ranging from 40s to 95s. Our method outperforms memory-free and memory-augmented baselines with a notable 30.6% gain over three real-world tasks. Real-world ablation studies confirm that event-driven keyframe selection outperforms uniform sampling with a 25% gain and latest history alternatives with a 16.7% gain, and that keyframe-aligned supervision provides independent performance gains beyond the memory mechanism alone.

Method

Event-driven keyframe memory framework. Keyframe detector module use robot states and visual observations to select task-relevant event keyframes, which are stored in a keyframe memory bank. The memory integration module retrieves the stored keyframes and fuses them with the current visual representation, enabling the VLA backbone to predict actions using both current observations and historical transition evidence. Right: module details for keyframe detection and memory interaction.

Keyframe Detection

Event-driven keyframe detection. The keyframe detector computes an event saliency score from robot motion, identifies candidate keyframes through causal peak detection, and applies a visual deduplication filter to keep only candidates associated with meaningful scene changes. Selected keyframes are stored, while visually redundant or non-transition frames are filtered out.

Memory Integration

Memory integration. The memory integration module encodes stored keyframes with the visual encoder, applies spatial pooling and temporal positional embeddings, retrieves keyframe memory tokens, and fuses them with the current visual representation through masked cross-attention and gated residual fusion before action prediction by the VLA backbone.
